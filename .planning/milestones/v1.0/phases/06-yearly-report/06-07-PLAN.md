Chart props (all components are pure presentational):

```typescript
import type { YearlyReport, YearlyReportBucket } from '@koinsight/common/dist/types/reports-api.js';

export function HeadlineCards({ totals }: { totals: YearlyReport['totals'] }): JSX.Element;

export function GenreBar({ data }: { data: YearlyReportBucket[] }): JSX.Element;
export function NationalityBar({ data }: { data: YearlyReportBucket[] }): JSX.Element;
export function DecadeHistogram({ data }: { data: YearlyReportBucket[] }): JSX.Element;
export function LanguagePie({ data }: { data: YearlyReportBucket[] }): JSX.Element;

export function CoverageBanner({ known, total, label }: {
  known: number; total: number; label: string;
}): JSX.Element;
```

Genre stacked bar shape: the response's `genre` is `[{key, count}]` (one row per genre, summed across all books). The "stacked bar" treatment per D-06 is one bar with multiple stacked segments, represent the data as a single-row dataset `[{ year: <yearLabel>, [genreKey]: count, ... }]` and pass `series=genreList.map(g => ({ name: g, color: ... }))` with `type="stacked"`. Use `dataKey="year"`. (Alternative: a vertical bar chart with one bar per genre, drop `type="stacked"` if Mantine 8.3.12's stacked single-row layout looks degenerate; either matches the requirement "stacked bar for genre breakdown".) Pick one consistent shape and document it inline.

Nationality bar: simple `[{ key, count }]` -> `BarChart data={data} dataKey="key" series={[{ name: 'count' }]} />`. The `Other` and `Unknown` keys are just additional entries.

Decade histogram: same shape as nationality.

Language pie: `PieChart data={[{ name, value, color }]}`, first use of PieChart in the repo, follow `@mantine/charts` PieChart docs.

Coverage banner: `<Text size="xs" c="dimmed" mt="xs">{label} known for {known} of {total} books read this year ({pct}%)</Text>`. `pct = total > 0 ? Math.round((known / total) * 100) : 0`.

Page composition: render order in `reports-yearly-page.tsx`:
1. `<YearNavigator />`
2. `<HeadlineCards totals={report.totals} />`
3. `<GenreBar data={report.genre} />` then `<CoverageBanner known={report.coverage.genre_known} total={report.coverage.total_books} label="Genres" />`
4. `<NationalityBar data={report.nationality} />` then `<CoverageBanner known={report.coverage.nationality_known} total={report.coverage.total_books} label="Nationality" />`
5. `<DecadeHistogram data={report.decade} />` then `<CoverageBanner known={report.coverage.publication_year_known} total={report.coverage.total_books} label="Publication year" />`
6. `<LanguagePie data={report.language} />` then `<CoverageBanner known={report.coverage.original_language_known} total={report.coverage.total_books} label="Original language" />`
